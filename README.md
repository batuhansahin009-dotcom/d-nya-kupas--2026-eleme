# Dünya Kupası 2026 Eleme Turnuvası

Bu proje Python ile geliştirilmiş bir Dünya Kupası 2026 eleme turnuvası takip sistemidir.

## Özellikler

- Maç sonucu girme
- Tur görüntüleme
- Sonraki tura geçme
- Şampiyonluk yolunu görüntüleme
- Gol krallığı tablosu
- Verileri JSON dosyasına kaydetme

## Kullanılan Teknolojiler

- Python
- JSON
- VS Code

## Projeyi Çalıştırma

Terminalde şu komutu çalıştırın:

```bash
python main.py
```
